🧠 Quiz Master: The Interactive Challenge
Welcome to my first dynamic JavaScript project! This is a web-based quiz game designed to test knowledge while providing a smooth, interactive user experience.

🚀 Key Features
Dynamic Questions: Powered by a JSON array, making it easy to scale with new content.

Smart Shuffling: Uses the Fisher-Yates algorithm to ensure options are randomized every time.

Real-time Feedback: Buttons flash Green for correct and Red for incorrect answers.

Penalty Scoring: Includes a -0.25 point deduction for wrong answers to keep the stakes high!

🛠️ The Tech Stack
HTML5 & CSS3: For the layout and button state transitions.

JavaScript (ES6+): Handling DOM manipulation, event listeners, and game logic.

GitHub Pages: For live hosting and deployment.

📈 What I Mastered
While building this, I moved beyond the basics by implementing state management (tracking the score and question index) and UI delays (using setTimeout) to create a better flow for the player.
